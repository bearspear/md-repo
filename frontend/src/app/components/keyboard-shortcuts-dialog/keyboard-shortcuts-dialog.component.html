Keyboard Shortcuts

Editor

Ctrl + B Bold text
Ctrl + I Italic text
Ctrl + S Save document
Ctrl + Z Undo
Ctrl + Shift + Z Redo

Find & Replace

Ctrl + F Open find & replace
Ctrl + H Open find & replace (with replace)
Enter Find next
Shift + Enter Find previous
Esc Close find & replace

Document

Ctrl + P Print document
Esc Close document
